## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js
- npm
- git

### Installation

1. Clone the repo:
```sh
git clone https://github.com/Choaib-ELMADI/portfolio.git
```

2. Install NPM packages:
```sh
npm install
```

3. Start the development server:
```sh
npm run dev
```
